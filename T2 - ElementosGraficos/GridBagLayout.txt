int posY = gridy = Fila

int tY = gridheight = Número de filas que ocupará el elemento
double peX = weightx  = peso en la columna
double peY = weighty = peso en la fila
fill = como se rellena el espacio vacío por el elemento = HORIZONTAL, VERTIAL, BOTH, NONE
int anc = anchor = la gravedad del elemento dentro de su celda = CENTER, PAGE_START, PAGE_END, LINE_START, LINE_END...
